Etiquetas semanticas:

Div - Es una etiqueta contenedora que no tiene semántica
Header - Es la cabezera de un sitio web o de una sección
Main - Define la sección principal del documetno, solo puede existir una etiqueta main por documento
Footer - Pié de página de un sitio web o de una sección
Article - Representa una sección de autocontenido (Por si solo se explica)
Aside - Representa un contenido complementario o secundario
Section - Representa una sección de contenido generico
Address - Representa una dirección

Etiquetas de Bloque VS Etiquetas de Linea

Div Es la etiqueta de bloque por Excelencia, una etiqueta de bloque es aquella qeu genera saltosde lineas con sus etiquetas hermanas
Span Es la etiqueta de Linea por Excelencia, una etiqueta de lineas solo ocupa el espacio que tiene su contenido Hola Soy otra span

Estilos CSS en HTML

Hay Ciertas formas de aplicar CSS a un documento HTML:

  • Linkear el documento CSS al docuemnto HTML:

    • <link rel="stylesheet" href="(DirecciónDelDocumento)">

  • Usar la etiqueta Style en el documento HTML:
    • Style {
       background-color: blue;
}
  • Usar el atributo Style en una etiqueta:
      • <h2 style='background-color:yellow;>

Scripts en HTML

Hay ciertas formas de escribir scripts en HTML:

  • Conectar un documento Javascript a un docuemnto HTML:

    • <script src="(DirecciónDelDocumento)"> </script>

  • Usar la etiqueta script

    • <script> </script>
